Introduction

This is a workshop put together by me to introduce Ontario Tech engineering students to object detection. It covers the following topics:

  1. How a single neuron (Perceptron) functions and is trained.
  2. How multiple neurons can be combined to form Feed-Forward Neural Networks.
  3. A basic overview of Convolutional Neural Networks
  4. An introduction to concepts important for object detection
  5. A demo where a Faster R-CNN model is used for inference

After the workshop has been presented, a filled out copy of the demo notebook will be made available. In the following days, I will post a write-up below summarizing the workshop itself and adding some content that I did not have time to cover.
