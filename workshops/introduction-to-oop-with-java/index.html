Introduction

This workshop is aimed at introducing OOP concepts to those who are new to programming. It will provide resources to help you setup a java development environment, and then quickly cover concepts integral to OOP. Finally we will do a quick example where we make a simple version of battleship.

Setup your Development Environment

To follow along with the workshop it will be beneficial to have a java environment set up. If you have time please read the rest of this section and set one up for yourself. If you are unable to, feel free to use an online Java compiler and runtime environment like this one.

Install the JDK and JRE

Java ships mainly in 2 distributions. There's the open source version which ships as OpenJDK and then the closed source version that is maintained and shipped by Oracle. I will be using OpenJDK version 11.

Feel free to use any version of Java and any IDE you are comfortable with. Ideally the demo will run on any of them, just make sure you have installed both the Java Development Kit (JDK) and the Java Runtime Environment (JRE). To setup java for your system follow these instructions:

  • Windows
  • MacOS
  • Ubuntu
  • Anything Else: You're on your own.

Setup an IDE

You can use any IDE you are comfortable with as long as it supports java (or you can just use a text editor + cli). I will personally be using IntelliJ IDEA. Please follow the instructions below to configure your IDE.

  • Visual Studio Code
  • IntelliJ IDEA
  • NetBeans
  • Eclipse
