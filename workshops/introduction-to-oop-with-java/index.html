2 - Object Oriented Programming

Object Oriented Programming is a programming paradigm that models computations using abstractions. This means we represent data or business logic using a collection of variables, grouped with functions that modify and fetch the underlying data in some way. We're going to follow a hands on approach to get an intuitive understanding of OOP in java.

First we are going to create an extremely simple representation of a car. We want it to consist of the following functionality:

Car {
    # attributes
    name
    top_speed
    position
    
    # actions
    getPosition()
    driveTo()
}

2.2 - Objects & Instances

Using the above defined class we can instantiate a new object. An object is an instance of a class, and its instance variables refer to the values that the object's attributes are taking on. This would look like:

public class Main {

    public static void main(String[] args) {
        double[] start_pos = {0.0, 0.0};
        CarTwo first_car = new Car("Civic", 100, start_pos);

        double[] new_pos = {1, 1};
        first_car.moveTo(new_pos, 150);
        first_car.getPosition()
    }
}

As you can tell, the position of the first_car object has been updated. Methods can either access, or mutate the instance variables of an object.

2.3 - Abstract Base Classes & Interfaces

We can take classes even further and create what are called abstract classes. They represent a way of producing hierarchical representation of information by associating objects. Lets create a generic vehicle class that Car can be based on.

First we make an interface. It acts like a contract that ensures an object meets a specification that codifies how an object is used.

public interface VehicleInterface {
    public void moveTo(double[] new_position, double speed);
    public String getPosition();
    public String getName();
}

Then we can define our matching Abstract Base Class.

public abstract class VehicleBaseClass implements VehicleInterface{

    public VehicleBaseClass(String vehicle_name, double max_speed, double[] start_position) {
        this.name = vehicle_name;
        this.position = start_position;
        this.top_speed = max_speed;
    }

    public String getPosition() {
    }

    public String getName() {
    }

    public abstract void moveTo(double[] new_position, double speed);
}
